Learn CSS Basics

Welcome to the styling world of web development! In this beginner's guide, you will learn the fundamentals of CSS, the language used to style and design web pages.

Think of HTML as the structure (the skeleton), and CSS as the style (the clothing and colors). Together, they bring websites to life!

What is CSS?

CSS (Cascading Style Sheets) controls the appearance and layout of HTML elements. It allows you to apply colors, fonts, spacing, and more, making web pages visually appealing.

CSS Syntax

A CSS rule consists of a selector and a declaration block:


selector {
  property: value;
}

Example:


p {
  color: blue;
  font-size: 16px;
}

Types of CSS

  1. Inline CSS: Applied directly to an HTML element using the style attribute.
  2. Internal CSS: Written within a <style> tag in the HTML <head>.
  3. External CSS: Written in a separate CSS file and linked to the HTML file using <link>.

Selectors

Selectors target HTML elements to apply styles. Common types include:

  • Element Selector (p, h1)
  • Class Selector (.classname)
  • ID Selector (#idname)
  • Universal Selector (*)
  • Group Selector (h1, h2, p)
  • Descendant and Child Selectors

Color and Background

Control color and background styles:


body {
  background-color: #f0f0f0;
}

h1 {
  color: #333333;
  background-image: url('background.png');
}

Text and Fonts

Customize text appearance with properties like:

Property Description
font-familySets the font
font-sizeAdjusts text size
colorSets text color
line-heightControls space between lines
text-alignAligns text

Box Model

The box model describes the structure of every HTML element:

  • Content
  • Padding
  • Border
  • Margin

        div {
        width: 200px;
        padding: 10px;
        border: 2px solid black;
        margin: 15px;
        }

Positioning & Display

Control how elements are placed and displayed:

  • Position: static, relative, absolute, fixed, sticky
  • Display: block, inline, inline-block, none
  • Visibility: visible, hidden

Flexbox

Flexbox is used for responsive layouts:


        .container {
        display: flex;
        justify-content: space-between;
        align-items: center;
        }

CSS Grid

CSS Grid provides two-dimensional layout control:


        .grid-container {
        display: grid;
        grid-template-columns: repeat(3, 1fr);
        gap: 10px;
        }

Media Queries

Media queries enable responsive design:


        @media (max-width: 768px) {
        body {
            background-color: lightblue;
        }
        }

Best Practices

  • Use external CSS files for maintainability.
  • Keep selectors specific and avoid overuse of IDs.
  • Use shorthand properties.
  • Organize your stylesheet with comments.
  • Minimize unnecessary use of !important.

CSS Units

CSS units are used to define sizes, margins, padding, and other properties. Common units include:

  • Absolute Units: px (pixels), pt (points), in (inches), cm (centimeters), mm (millimeters)
  • Relative Units: % (percentage), em (relative to font size), rem (relative to root font size), vw (viewport width), vh (viewport height)

        .container {
        width: 80%; /* Relative unit */
        padding: 20px; /* Absolute unit */
        font-size: 1.2rem; /* Relative to root font size */
        }

CSS Variables

CSS variables (custom properties) allow you to store reusable values, making your stylesheets more maintainable. They are defined using --variable-name and accessed with var().


        :root {
        --primary-color: #85573b;
        --secondary-color: #ab0a0a;
        }

        body {
        background-color: var(--primary-color);
        color: var(--secondary-color);
        }

CSS Transitions

CSS transitions allow you to smoothly change property values over a specified duration. They are commonly used for hover effects and animations.


        .button {
        background-color: #85573b;
        color: white;
        padding: 10px 20px;
        transition: background-color 0.3s ease, transform 0.3s ease;
        }

        .button:hover {
        background-color: #ab0a0a;
        transform: scale(1.1);
        }
