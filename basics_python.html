Introduction to Python

Python is a versatile, high-level programming language known for its simplicity and readability. It's widely used in web development, data analysis, artificial intelligence, and more.

Variables and Data Types

In Python, variables store data values. Common data types include integers, floats, strings, and booleans:


x = 5        # Integer
y = 3.14     # Float
name = "John" # String
is_valid = True # Boolean

Operators

Python supports arithmetic, comparison, logical, and assignment operators:


result = 10 + 5  # Addition
is_equal = 10 == 5  # Comparison
logical_and = True and False  # Logical Operator

Conditional Statements

Conditional statements allow you to make decisions in your code:


if x > 0:
    print("Positive number")
elif x == 0:
    print("Zero")
else:
    print("Negative number")

Loops

Loops are used to repeat code until a condition is met:


# For loop
for i in range(5):
    print(i)

# While loop
count = 0
while count < 5:
    print(count)
    count += 1

Functions

Functions are blocks of reusable code:


def greet(name):
    print(f"Hello, {name}")

greet("Alice")

Classes and Objects

Python supports object-oriented programming through classes and objects:


class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

person1 = Person("Alice", 25)
print(person1.name)

Exception Handling

Handle errors using try-except blocks:


try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")

Modules and Libraries

Python has numerous built-in and external libraries. Importing a module is easy:


import math
print(math.sqrt(16))
